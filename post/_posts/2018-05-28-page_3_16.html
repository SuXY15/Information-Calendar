本科生、研究生退课时间安排（17-18春）Course-Dropping

选课系统分为中、英文两种语言，同学们可选择其中一种语言进入选课系统。

选课系统适用于 win7 操作系统，IE浏览器 9-11 版本均可。使用其他版本的操作系统或浏览器进行选退课操作，有可能出现异常情况导致选退课失败。若出现同学自己的电脑环境不支持选课系统的情况，请及时到中央主楼开放实验室机房进行选课操作。
Enter course registration system by logging into: http://academic.tsinghua.edu.cn. You can select English /Chinese language to access to course registration system. The system supports Win7, IE 9-11 browser. Other system or web browser may result in unknown error and failure on adding/deleting course. If problems found on personal computer, please go to the university lab on main building to do course-registration.

使用校外网络选课的同学，请务必按照附件中的操作步骤，安装VPN客户端后方可进行选课操作。
If you are off campus, you should set a VPN client to visit the intranet to do any course registration.
关于校外选课的网络操作说明.doc    Off-Campus Network Settings for Course Registration.pdf

电话咨询：选课问题，请致电注册中心 62794721
          培养方案方面的问题，请致电研究生院培养办 62795078

退课登录方法：在清华大学教学门户（ http://academic.tsinghua.edu.cn ）用户登录后，于“教学信息”页面的“本学期课程”栏目中，点击【进入选课】按钮，即可登陆选课系统，再进入退课界面进行操作。

进入选课系统的操作方法（图文）：jwcbg/detail_cat.jsp?boardid=57&seq=5005

退课阶段

退课（第二阶段）：2018 年 4 月 29 日(星期日) 8:00 -- 5 月 16 日(星期三) 16:00
  前八周已结束的课程、本科生体育课、部分外文系课程（课程号为0064或1064开头）不允许退课。

Course Drop II:  2018-04-29 (Sun.) 8:00 ～ 2018-05-16 (Wed.) 16:00
Students cannot drop: Courses finished in first 8 weeks, PE courses and courses whose number begin with 0064/1064 (general language courses given by Department of Foreign Languages and Literatures).  The dropped course (except those begin from or after 9th academic week)  will be marked with “W”（Withdrew）on transcript.


【特别说明】

1、退课只在网上进行，网上退课结束后没有其他形式的退课。

2、退课阶段所退课程在成绩单上记录“W”（Withdrew），但第九周及以后开课课程的退课不记“W”，会在退课结束后统一取消“W”记录。

3、退课阶段，研究生不可维护个人培养计划。

4、使用校外网络查询或操作退课，必须安装VPN客户端，否则将不能正常显示课程。不可使用手机网络查询退课情况。
