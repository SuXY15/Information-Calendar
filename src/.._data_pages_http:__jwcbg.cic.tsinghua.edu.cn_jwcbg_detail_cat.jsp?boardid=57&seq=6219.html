1．具有清华大学学籍的大一、大二全日制在校本科生； 2. 主修专业学习成绩优良，学有余力； 3．未修读其他第二学士学位/辅修专业； 4. 符合相应的第二学士学位/辅修专业开设院系的其他申请条件。 注： 每位学生只能申请一个专业。 双培计划学生可以申请辅修，但不可以申请二学位。 二、招生工作日程 1. 5月7日8:00～5月10日24：00点，学生在网上申请报名。按照二学位/辅修开设院系要求提交申请材料。 2. 5月11日～5月14日，主修院系对报名学生进行审核。 3. 5月21日前，二学位/辅修院系通过系统向教务处提交考核结果。 4. 5月25日，教务处在教务公告上公示拟录取名单。 三、学生网上申请方法 登录“信息门户―学习―快捷导航―公告”下面的二级菜单“二学位辅修申请”。 点击“二学位辅修申请”进去后有三条链接： 1、 网上申请二学位/辅修 在这里可查看院系的招生计划和实施方案，并填写申请表，如有其他相关材料可以附件形式上传（如果多于一个文件，可以打包成压缩文件上传）。学生在网上填写申请表后可以选择提交或暂存，选择提交不可以再修改，选择暂存可以修改（选择暂存的同学在5月10日24点申请截止时间前一定不要忘记提交申请表）。二学位/辅修申请以网上提交申请的学生名单为准，暂存的申请无效。 学生在填写申请表时，一定要正确填写手机号码、Email等个人信息。 2、 查看二学位/辅修申请记录 在这里可查看个人的申请表。 3、 查看二学位/辅修录取结果 二学位/辅修名单公示后，学生可以在这里查看自己的录取情况。 注意：如果使用低版本的IE，页面中的数据可能会不显示。建议用IE11非兼容模式、谷歌、火狐浏览器等。 网申过程中如有系统技术问题请咨询信息化技术中心，电话：62771940。 如需了解某个专业，请和相关二学位/辅修专业开设院系联系，联系电话见附件3。 教务处联系电话：62772337（二学位），62784319（辅修） 另外，学生学习与发展指导中心提供“一对一咨询”项目，可帮助同学解决二学位、辅修或其他问题，电话：62792453。“一对一咨询”预约指南网址如下：
